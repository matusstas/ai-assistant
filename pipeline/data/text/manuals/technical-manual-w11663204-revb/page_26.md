# DIAGNOSTICS AND TROUBLESHOOTING (CONT.)

## For Service Technician Use Only
### Manual Overview Test Mode

Pressing the START button will activate/deactivate each output. When the output is activated, the corresponding Status LEDs will flash. See page 2-6 to access Manual Overview Test Mode.

**IMPORTANT:** Lid must be closed and locked to perform SPIN and AGITATE tests.

### OUTPUT
**Output Details**
**NOTE:** Outputs will time-out after 5 minutes.

| OUTPUT          | Status LEDs |
|-----------------|-------------|
|                 | SOAK/WASH   |
|                 | RINSE       |
|                 | FINAL SPIN  |
|                 | DONE        |
|                 | LID LOCKED  |
| **LID LOCK**    |             |
| Lock and unlock the lid. | 1           |
| **NOTES:**      |             |
| - When lock is enabled, the “Lid Locked” LED will turn ON. Will only lock when lid is closed. |
| - If lid is not closed, washer will flash status LEDs on and off. |
| - If lid does not lock or unlock, go to **TEST #8: Lid Lock.** |

| **COLD VALVE**  |             |
| Turns ON and turns OFF cold water valve.    | 2           |
| - If valve does not turn on, go to **TEST #2: Valves.** |

| **HOT VALVE**   |             |
| Turns ON and turns OFF hot water valve.     | 2           |
| - If valve does not turn on, go to **TEST #2: Valves.** |

| **DRAIN**       |             |
| Turns ON and turns OFF the drain pump.      | 2           |
| - If pump does not turn on, go to **TEST #7: Drain Pump.** |

| **LOW SPIN**    | To perform test, lid must be closed and locked. | 8 | 1 | ON |
| Spins basket from 0 to 500 RPM. |
| **NOTE:** Allow up to 15 seconds for shifter to reposition. |
| **IMPORTANT:** To activate Low Spin, RPM must read “0” and lid must be closed with lid lock enabled. |
| - If lid is not closed, status LEDs will flash on and off. |
| **IMPORTANT:** Water in tub must be drained before test. |
| - If motor does not spin, go to **TEST #3a: Drive System – Shifter and TEST #3b: Drive System – Motor.** |

| **HIGH SPIN**   | To perform test, lid must be closed and locked. | 8 | 1 | ON |
| Spins basket from 0 to maximum RPM. |
| **NOTE:** Allow up to 15 seconds for shifter to reposition. |
| **IMPORTANT:** To activate High Spin, RPM must read “0” and lid must be closed with lid lock enabled. |
| - If lid is not closed, status LEDs will flash on and off. |
| **IMPORTANT:** Water in tub must be drained before test. |
| - If motor does not spin, go to **TEST #3a: Drive System – Shifter and TEST #3b: Drive System – Motor.** |
