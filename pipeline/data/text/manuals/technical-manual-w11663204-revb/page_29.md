# DIAGNOSTICS AND TROUBLESHOOTING
## For Service Technician Use Only

| Problem                          | Possible Cause                          | Checks and Tests                                                                             |
|----------------------------------|----------------------------------------|----------------------------------------------------------------------------------------------|
| CYCLE TIME LONGER THAN EXPECTED | Oversuds.                              | 1. Verify use of HE detergent. <br> 2. Excessive detergent usage.                          |
|                                  | Off balance.                           | 1. Load is off balance. <br> 2. Balance ring water leak.                                   |
|                                  | Draining slowly.                       | Check for pump or drain hose obstructions.                                                  |
|                                  | Water pressure drop.                  | Results in longer fill time.                                                                 |
|                                  | Friction or drag on drive.            | Check motor and bearings; check for clothes between tub and basket.                          |
|                                  | Weak suspension.                       | Basket should not bounce up and down more than once when pushed.                            |
| POOR WASH PERFORMANCE            | Oversuds.                              | 1. Verify use of HE detergent. <br> 2. Excessive detergent usage.                           |
|                                  | Load is tangling.                     | 1. Washer not loaded properly. <br> 2. Perform washer calibration.                           |
|                                  | Incorrect water level.                | 1. Perform washer calibration. <br> 2. See TEST #2: Valves. <br> 3. See TEST #6: Water Level. |
|                                  | Clothes wet after cycle is complete   | 1. Overloaded washer. <br> 2. Oversuds (see above). <br> 3. Items caught in tub sump. <br> 4. Weak suspension. <br> 5. Shifter not moving into position (see TEST #3a: Drive System – Shifter). <br> 6. Cold/Rinse water > 105°F (41°C). <br> 7. See TEST #7: Drain Pump. |
|                                  | Load not rinsed.                      | 1. Check proper water supply. <br> 2. Not using HE detergent. <br> 3. Washer not loaded properly. <br> 4. Shifter not moving into position (see TEST #3a: Drive System – Shifter). <br> 5. See TEST #2: Valves. |
|                                  | Not cleaning clothes.                 | 1. Washer not loaded properly. <br> 2. Not using HE detergent. <br> 3. Not using correct cycle. <br> 4. Shifter not moving into position (see TEST #3a: Drive System – Shifter). |
|                                  | Fabric damage.                        | 1. Washer overloaded. <br> 2. Bleach added incorrectly. <br> 3. Sharp items in tub.        |
|                                  | Wrong option or cycle selection.     | Select option or cycle appropriate for the load.                                            |
