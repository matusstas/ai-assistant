# 5. MALFUNCTION CODES AND EXPLANATIONS

## E80
- **Define:** Communication error alarm.
- **Reasons:** There is no communication between main control board and display panel.

| Malfunction code | Root Reason                                                 | Possible cause                                                          |
|------------------|------------------------------------------------------------|-------------------------------------------------------------------------|
| E80              | There is no communication between main control board and display panel | Check the communicate cable connector between main control board and display panel <br> Main control board communication terminal failure <br> Display panel communication terminal failure |

## Check Procedure:

1. **Check the communicate cable connector between main control board and display panel**
   - Yes: It’s recommended that user need to confirm the communication cable connector is fixed.
   - No: 
     2. **Whether main control board communication terminal failure?**
        - Yes: It’s recommended that the user need to repair or change the main control board.
        - No: 
          3. **Whether display panel communication terminal failure?**
             - Yes: It’s recommended that the user need to repair or change the main control board.
