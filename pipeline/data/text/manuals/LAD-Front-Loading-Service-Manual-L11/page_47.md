# 5. MALFUNCTION CODES AND EXPLANATIONS

## E64

**Check Procedure:**

- Check whether the harness connecting the wire to the motor is installed in place, and whether the harness connecting the wire to the main PCB is not installed in place.
  - **No** → Adjust housing of harness
  - **Yes** 
    - Replace the motor and check whether the problem can be solved.
      - **Yes** → The problem is motor defect
      - **No** 
        - Replace the main PCB and check whether the problem can be solved.
          - **Yes** → The problem is main PCB defect
          - **No** 
            - Replace the harness and check whether the problem can be solved.
              - **No** → The problem harness defect
              - **Yes**

---

The biggest possibility of this problem is:
1. The harness and motor or harness and main PCB are not properly assembled.
2. The motor is defective. It is less likely that other main PCB are defective and there is a broken wire somewhere.
