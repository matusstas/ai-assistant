# 5. MALFUNCTION CODES AND EXPLANATIONS

## E21
**Check Procedure:**

- **Whether the drain hose kinked**
  - Yes ➔ Check the drain hose and straighten the hose
  - No ➔
  
- **Whether the drain pump Filter is blocked**
  - Yes ➔ Check & Clean Pump Filter. This drain filter should be cleaned once a month
  - No ➔

- **Whether the height of the drain hose is beyond 1m**
  - Yes ➔ Adjust it according to the installation part of user manual
  - No ➔

- **Whether the wiring connection is well**
  - Yes ➔ Reliable connection
  - No ➔ Change the drain pump

- **Whether the drain pump is well**
  - Yes ➔
  - No ➔ Change the water level sensor

- **Whether the water level sensor is well?**
  - Yes ➔ Check the wire connector and harness before replacing PCB.
  - No ➔ Change the water level sensor
