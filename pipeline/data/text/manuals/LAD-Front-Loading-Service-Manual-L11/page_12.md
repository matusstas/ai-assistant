```markdown
## 5. MALFUNCTION CODES AND EXPLANATIONS

### E10
- **Define:** Water injecting problem during wash cycle.
- **Reasons:** The root reason is the water level doesn’t change within 3 minutes.

| Malfunction code | Root Reason                             | Possible cause                                                                                                                                         |
|------------------|-----------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| E10              | The water level doesn’t change within 3 minutes. | If the washer is filling very slowly, the water pressure from the house might be too low. The water pressure must be greater than 0.1 MPa.          |
|                  |                                         | Water faucet is not turned on or the screens on the hoses are blocked.                                                                               |
|                  |                                         | The screens inside the water inlet valve are blocked or the water inlet valve is damaged.                                                            |
|                  |                                         | Something is wrong with water level sensor.                                                                                                          |
```
