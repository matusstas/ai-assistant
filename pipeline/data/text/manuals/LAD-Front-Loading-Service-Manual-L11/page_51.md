# 5. MALFUNCTION CODES AND EXPLANATIONS

## E80

### Check Procedure:

#### Step 1: Check the connector between main control board and display panel
1. If the communication cable connector is loosen the E80 warning will occur when the washing machine is running. It’s recommended that user need to confirm the communication cable connector is fixed. (Fig.1)

#### Step 2: Main control board communication terminal failure
1. Check whether the communication terminal is damaged, if it’s damaged the E80 warning will occur. It’s recommended that the user need to repair or change the main control board. (Fig.2)

#### Step 3: Display panel communication terminal failure
1. Check whether the communication terminal is damaged, if it’s damaged the E80 warning will occur. It’s recommended that the user need to repair or change the main control board. (Fig.3)
